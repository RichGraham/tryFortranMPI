====Introduction====
A basic mpi function with tests in Fortran.

====Local setup====
UoN HPC: run - source UoN_HPC_SetUp.sh

====Running====
build_and_run.x - builds and runs the basic code
build_and_run_tests.x - builds and runs the tests on a small number of processor
manyProcessorTests.x - runs the tests on a progressively increasing number of processors
